 # 💡Futher Reading
## 1. 1bit(1비트)
- 1비트 : 전기 스위치 1개(0과 1로 표현) / ***표현의 최소 수준***
- **1바이트(byte)** 
    - 8개 비트를 하나로 묶음 / 기억공간의 최소 단위 / 기억공간 관리할 때 1byte단위로 관리 
    - **영문자 한 글자**가 저장될 수 잇는 메모리 크기(한글 한 글자는 2byte 필요)
- 2의 32 제곱 = 32bit = **4GB(기가바이트)**
## 2. 진법변환
- 1비트 : 2진수 한 자리 숫자
- 4비트 : 16진수 한 자리 숫자
    - ex : 0x41(16) > 4(0100), 1(0001) (bin) = 8bit = 1byte
